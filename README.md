# 🎯 Simple Quiz App

A responsive and interactive **Quiz Application** built using **HTML, CSS, and JavaScript**. The app presents multiple-choice questions, provides instant feedback, tracks the user's score, and allows the quiz to be restarted.

## 📚 How It Works

1. The quiz starts with the first question.
2. Select one of the available answers.
3. The correct answer is highlighted.
4. Click **Next** to continue.
5. After the final question, your score is displayed.
6. Click **Play Again** to restart the quiz.
